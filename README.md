vimconfig
=========
My Vim config files

OS X and Linux Users
--------------------
    git clone git://github.com/ianoxley/vimconfig.git
    cd
    ln -s /path/to/vimconfig/vimrc ~/.vimrc

Windows Users
-------------
    git clone git://github.com/ianoxley/vimconfig.git 
    cd \path\to\home\directory
    mklink _vimrc \path\to\vimconfig\vimrc
